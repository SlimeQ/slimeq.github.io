# Portfolio

Personal portfolio site for Quincy Campbell.

## Deploy to GitHub Pages

1. Create a new repo named `slimeq.github.io` (or `portfolio`)
2. Push this folder's contents
3. Go to Settings → Pages → Source: Deploy from branch (main)
4. Your site will be live at `https://slimeq.github.io/`

## Local Preview

Just open `index.html` in a browser.
